
# new-generator.html (بدون أي تغيير للموقع الحالي)

هذه صفحة مستقلة لتوليد ملفات الاختبارات وروابطها، تعمل مع موقعك كما هو تماماً.

## ماذا تفعل؟
- تولّد ملف JSON لكل اختبار باسم عشوائي مثل `q7ab12cd.json`.
- تعطيك الرابط الجاهز للطلاب بصيغة: `student.html?id=q7ab12cd`.
- لا تعدّل أي ملف آخر.

## طريقة الاستخدام
1) ارفع الملف `new-generator.html` إلى جذر المستودع (نفس مستوى index.html).
2) افتحها عبر الرابط: `.../new-generator.html`.
3) اكتب العنوان والمدة والصق الأسئلة (JSON)، ثم اضغط "توليد الملف والرابط".
4) سيتم تنزيل ملف مثل `q7ab12cd.json` على جهازك.
5) ارفعه يدوياً إلى مجلد `data/quizzes/` في نفس المستودع.
6) استخدم الرابط الذي يظهر لك للطلاب. بمجرد رفع الملف للمجلد الصحيح سيعمل الرابط فوراً.

> لا حاجة لتعديل student.html أو أي صفحة أخرى.
