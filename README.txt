طريقة الاستخدام (رفع فقط):
- ارفع الملف nav-strip.js إلى نفس مجلد موقعك (حيث index.html).
- إذا لديك ملف JavaScript عام يتم تحميله في كل الصفحات (مثل app.js)، انسخ محتوى nav-strip.js بداخله في آخر الملف.
- أو (إذا تقبل إضافة وسم سكربت لاحقًا) أضف: <script src="./nav-strip.js" defer></script> قبل </body>.

مفعول الملف:
- يزيل الأزرار "التعليمات السريعة" و"تواصل مع الإدارة" و"لوحة التحكم" من كل الصفحات.
- يبقي زر "الرئيسية" كما هو في المنتصف، ولا يغير أي عنصر أو تنسيق آخر.
